# Data Directory Structure and Contents

| Folder | Description |
| --- | --- |
| **raw_reports**  | Contains html and json files containing fastp report summaries. Reports describe general, before filtering, after filtering, and filtering results. |
| **trimmed_reads**  | Files not uploaded to keep GitHub appropriate size and function. Data files can be accessed through request, and will be uploaded to data specific repository at the end of project. |
